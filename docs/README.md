# Mridangam tutorial

## Sloka to recite.

`Namo Jaganatha suthaya`

`Namo Mridangaya namaha`

`Lavanya rasa Madhuri`

`Sahasra guna samyuktham`

`Namo Mridangaya Namo Namaha`

`Namo Baladevaya Namo Namaha`

## Strokes

|  Symbols          |      Description                   |
|-------------------|------------------------------------|
|  ~<b>llll</b>~    | Small Side, 4 Finger Closed Stroke |
|  ~<b>lll</b>~     | Small Side, 3 Finger Closed Stroke |
|  ~<b>l</b>~       | Small Side, 1 Finger Closed Stroke |
|  <b>l</b>         | Small Side, 1 Finger Open Stroke   |
|  <b>lll</b>       | Small Side, 3 Finger Open Stroke   |
|  <b>llll</b>      | Small Side, 4 Finger Open Stroke   |
|  <b>BC</b>        | Big side close with 4 fingers      |
|  <b>BO</b>        | Big side Open with 4 fingers       |
